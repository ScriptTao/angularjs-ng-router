图片功能演示

图片标签中，使用src指定图片的路径。使用示例如下： 


    <img src="../../images/natrual.jpg" alt="提示文本" class="img-rounded" style="width:100px; height:100px">
    <img src="../../images/natrual.jpg" alt="提示文本" class="img-circle" style="width:100px; height:100px">
    <img src="../../images/natrual.jpg" alt="提示文本" class="img-thumbnail" style="width:100px; height:100px">

Javascript方法如下： 


    $(document).ready(function () {
        $("#setImg").click(function () {
            $("img").attr("src", "../../images/logo_right.png")
        });


        $("#reset").click(function () {
            $("img").attr("src", "../../images/natrual.jpg")
        });
    });
