spinner功能演示

实现spinner的功能，更多示例请参考 http://api.jqueryui.com/1.11/spinner/

基本使用方法

在html文档的header区域引入样式文件，在body底部引入javascript文件。请注意引入的文件路径


<link rel="stylesheet" href="../../bootstrap-3.3.7/css/bootstrap.min.css">
<link rel="stylesheet" href="../../jquery-ui-1.12.1/jquery-ui.min.css">
<link rel="stylesheet" href="../../css/ustb.css">

<script src="../../jquery-3.2.1/jquery-3.2.1.min.js"></script>
<script src="../../bootstrap-3.3.7/js/bootstrap.min.js"></script>
<script src="../../jquery-ui-1.12.1/jquery-ui.min.js"></script>
<script src="../../js/q.js""></script">
<script src="../../js/ustb.js"></script>

使用input创建slider元素，并创建显示slider值的span


    <div class="well">
        <div>
            <label for="spinner">选择一个值：</label>
            <input id="spinner" name="value">
        </div>
        <div style="margin-top: 10px;">
            <button class="btn btn-primary" id="getVal">获取值</button>
            <button class="btn btn-primary" id="setVal">设置值为5</button>
        </div>
    </div>

在文档加载完毕后，创建spinner控件


    $(document).ready(function () {
        var spinner = $("#spinner").spinner({
            min: 1,
            max: 10,
            step: 1,
            start: 6
        });

        $("#getVal").click(function () {
            alert($("#spinner").spinner("value"));
        });

        $("#setVal").click(function () {
            $("#spinner").spinner("value", 5);
        });
    });
