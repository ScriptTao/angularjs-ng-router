单选框功能演示

单选框使用input标签实现，type设置为checkbox 


    <div class="checkbox">
        <label>
            <input type="checkbox" value="checkbox1">
            选项1
        </label>
    </div>
    <div class="checkbox">
        <label>
            <input type="checkbox" value="checkbox2">
            选项2
        </label>
    </div>
    <div class="checkbox disabled">
        <label>
            <input type="checkbox" value="checkbox3" disabled>
            选项3已经被禁用
        </label>
    </div>

Javascript方法如下： 


    $(document).ready(function () {
        $("#getText").click(function() {
            var text = "";
            $("input[type='checkbox']:checked").each(function(index, elem) {
                text += $(elem).val() + ", "
            });
            alert(text);
        });

        $("#setText").click(function() {
            $("input[type='checkbox'][value='checkbox2']").prop("checked", true);
        });
    });
