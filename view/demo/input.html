文本输入框

文本输入框功能演示

输入框采用input标签实现，input标签中的placeholder为输入提示语句。其class一般设置为form-control。 如果需要设置其他样式，可以使用style直接设置，或者添加其他css类别。

input输入框通常和form表单一起配合使用。详细form样式可以参考：http://www.runoob.com/bootstrap/bootstrap-forms.html 


    <input id="input" class="form-control" type="text" placeholder="请输入内容" />

Javascript方法如下： 


    $(document).ready(function () {
        $("#getText").click(function() {
            var text = $("#input").val();
            alert(text);
        });

        $("#setText").click(function() {
            $("#input").val("新的数值");
        });

        $("#setStyle").click(function() {
            $("#input").css("color", "red");
            $("#input").css("font-size", "30px");
        });

        $("#setDisable").click(function() {
            var disabled = $("#input").is(":disabled");
            $("#input").attr("disabled", !disabled);
            $(this).text(!disabled ? "启用" : "禁用");
        });

        $("#reset").click(function () {
            $("#input").val("");
            $("#input").css("color", "black");
            $("#input").css("font-size", "14px");
        });
    });
