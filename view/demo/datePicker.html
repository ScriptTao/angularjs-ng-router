日期选择

本例集成开源日期时间选择控件，详细使用请参考 控件官网

基本使用方法

在html文档的header区域引入样式文件，在body底部引入javascript文件。请注意引入的文件路径


<link rel="stylesheet" href="../../bootstrap-3.3.7/css/bootstrap.min.css">
<link rel="stylesheet" href="../../bootstrap-datetimepicker/css/bootstrap-datetimepicker.min.css">
<link rel="stylesheet" href="../../highlight/styles/rainbow.css">
<link rel="stylesheet" href="../../css/ustb.css">

<script src="../../jquery-3.2.1/jquery-3.2.1.min.js"></script>
<script src="../../bootstrap-3.3.7/js/bootstrap.min.js"></script>
<script src="../../bootstrap-datetimepicker/js/bootstrap-datetimepicker.js"></script>
<script src="../../echarts/echarts-3.8.0.js"></script>
<script src="../../js/q.js""></script">
<script src="../../js/ustb.js"></script>

创建input作为日期选择控件


    <type="text" class="form-control" value="2017-12-28" style="width:200px;" />

调用生成控件的方法生成控件


    $("#btnDate").click(function() {
        $("input").datetimepicker('remove');
        $("input").datetimepicker({
            format: "yyyy-mm-dd",
            autoclose: true,
            minView: 'month',
            language: "cn"
        });
    });

    $("#btnTime").click(function() {
        $("input").datetimepicker('remove');
        $("input").datetimepicker({
            format: "yyyy-mm-dd hh:ii:ss",
            autoclose: true,
            maxView: "month",
            language: "cn"
        });
    });
