时间插件值读写

示例：


弹窗插件使用只需在 input 中加入onclick="WdatePicker({dateFmt:'yyyy-MM-dd HH:mm:ss',isShowClear:true});" onchange="" 即可 JS中无需加入代码

代码

HTMLdatePicker 
        
            <input type="text" onchange="" ng-model="subdataTime" onclick="WdatePicker({dateFmt:'yyyy-MM-dd HH:mm:ss',isShowClear:true});">
            <button ng-click="getdate()">获取选中时间</button>
            <button ng-click="setdate()">设置相应时间</button>
JS 
        
            monitor.controller('datePickerCtrl', function ($scope, $state, $stateParams, ngDialog , $interval) {
    $scope.subdataTime = '';
    $scope.getdate = function () {
        alert($scope.subdataTime);
    }
    $scope.setdate = function () {
        $scope.subdataTime = '1919-06-11 15:32:54'
    }
})
