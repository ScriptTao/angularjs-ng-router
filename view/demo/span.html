基本使用方法

在html中有三种显示固定文本的标签，文本的属性由css确定。css有两种指定方式，一种为直接写在style属性中。 另外一种则制定css文件，在属性中使用class指定css类别。

直接使用style指定如下： 


    <div>
        <label id="label" style="color:red;width:100px;height:50px; font-size:16px;">标签文字</label>
    </div>
    <div>
        <span id="span" style="color:blue;width:100px;height:50px; font-size:20px;">span文字</span>
    </div>
    <div>
        <p id="p" style="color:orange;width:300px;height:50px; font-size:24px;text-indent: 2em;">段落文字</p>
    </div>

使用css样式时，示例如下： 


    <div>
        <label class="label">标签文字</label>
    </div>
    <div>
        <span class="span">span文字</span>
    </div>
    <div>
        <p class="p">段落文字</p>
    </div>

css文件如下：


    .label {
        color:red;
        width:100px;
        height:50px;
        font-size:16px;
    }
    .span {
        color:blue;
        width:100px;
        height:50px;
        font-size:20px;
    }
    .p {
        color:orange;
        width:300px;
        height:50px;
        font-size:24px;
    }

在上述样式中，p表示段落，其css样式中有 text-indent 表示段落开头空几个字符。

除了使用静态的css方法外，还可以使用javascript动态改变元素的属性。 


    $(document).ready(function () {
        $("#getText").click(function() {
            var text = "label: " + $("#label").text() + "\n"
                  + "span: " + $("#span").text() + "\n"
                  + "p: " + $("#p").text();
            alert(text);
        });

        $("#setText").click(function() {
            $("#label").text("新设置的label");
            $("#span").text("新设置的span");
            $("#p").text("新设置的p");
        });

        $("#setStyle").click(function() {
            $("#label").css("color", "black");
            $("#span").css("font-size", "30px");
            $("#p").css("text-indent", "0");
        });

        $("#reset").click(function () {
            $("#label").text("标签文字")
                 .css("color", "red").css("width", "100px")
                 .css("height", "50px").css("font-size", "16px");

            $("#span").text("span文字")
                .css("color", "blue").css("width", "100px")
                .css("height", "50px").css("font-size", "20px");

            $("#p").text("段落文字")
                .css("color", "orange").css("width", "300px")
                .css("height", "50px").css("font-size", "24px")
                .css("text-indent", "2em");
        });
    });
